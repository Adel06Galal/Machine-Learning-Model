# 📌 Overview

**Machine Learning Professional GUI Application** is a modern desktop platform that simplifies the complete machine learning pipeline through an interactive graphical interface.

The application allows users to:

- Import datasets
- Analyze data
- Apply preprocessing techniques
- Train multiple ML models
- Evaluate performance

---

# ✨ Features

# 📁 Dataset Management

Complete dataset workflow:

✅ Upload CSV files  
✅ Interactive data preview  
✅ Reset sample dataset  
✅ Export processed datasets  


---

# 📊 Data Visualization

Explore datasets visually using:

- Scatter Plot
- Histogram
- Box Plot
- Correlation Heatmap

- Matplotlib
- Seaborn

## Missing Values Handling

Supported techniques:

- Mean Imputation
- Median Imputation
- KNN Imputer
- Iterative Imputer


---

## 🔤 Feature Encoding

Convert categorical features:

- Label Encoding
- One Hot Encoding


---

## 📏 Feature Scaling

Normalization techniques:

- StandardScaler
- MinMaxScaler


---

## 📉 Dimensionality Reduction

Implemented:

- PCA (Principal Component Analysis)

## ⚖️ Data Balancing

Handle imbalanced datasets using:

- SMOTE
- Synthetic Minority Oversampling Technique
