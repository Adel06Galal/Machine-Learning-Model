# 🤖 Machine Learning Professional GUI Application

A modern **Machine Learning Desktop Application** built with **Python, CustomTkinter, Scikit-Learn, and Imbalanced-Learn**.

This project provides a complete graphical interface for data preprocessing, visualization, model training, and evaluation without writing ML code manually.

---

## ✨ Features

### 📁 Dataset Management
- Upload CSV datasets
- Preview dataset using interactive table
- Reset sample dataset
- Export processed dataset

---

### 📊 Data Visualization
Generate different visualizations:

- Scatter Plot
- Histogram
- Box Plot
- Correlation Heatmap

Powered by:
- Matplotlib
- Seaborn

---

### ⚙️ Data Preprocessing

The application supports:

#### Missing Values Handling
Using:

- Mean Imputation
- Median Imputation
- KNN Imputer
- Iterative Imputer


#### Encoding

Convert categorical data using:

- Label Encoding
- One Hot Encoding


#### Feature Scaling

Available:

- StandardScaler
- MinMaxScaler


#### Dimensionality Reduction

- PCA (Principal Component Analysis)


#### Data Balancing

Using:

- SMOTE (Synthetic Minority Oversampling Technique)
